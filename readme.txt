Плагин для виджетов ProdaLet для вашего сайта на WordPress.

== Description ==

Данный плагин позволит вам легко вставить код ProdaLet на ваш сайт. При этом вам не нужно редактировать файлы вашей темы, и сервис продолжит работать даже при смене вашей активной темы.

После активации плагина зайдите в раздел Настройки - ProdaLet для того, чтобы вставить код установки. Учтите, что для редактирования кода счетчика, вам необходимы права администратора на сайте WordPress (или супер-администратора в режиме Multisite).

== Installation ==

1. Распакуйте архив с плагином
2. Загрузите папку с плагином в директорию `/wp-content/plugins/` на вашем сервере
3. Активируйте плагин ProdaLet в разделе Плагины в панели администрирования WordPress
4. Зайдите в раздел Настройки - ProdaLet для вставки кода вашего счётчика
